# SARTORIAL - Модное второе дыхание

Минималистичный веб-сайт для Telegram мини-приложения, которое продает б/у вещи премиум-класса. Имеет чистый дизайн, вдохновленный модными домами высокой моды, такими как Cévennes и Balenciaga.

## Особенности

- **Минималистичный дизайн**: Чистый, элегантный интерфейс с вниманием к типографике и пространству
- **Защищенный вход для администраторов**: Логин (sartorial) и пароль (favicon) для доступа к панели управления
- **CRUD функциональность**: Возможность создавать и удалять объявления
- **Интеграция с Telegram**: Прямое подключение к продавцам через Telegram
- **Адаптивный дизайн**: Отлично работает на всех размерах устройств
- **Анимации**: Плавные анимации и переходы для улучшенного UX
- **Простота развертывания**: Готов к работе на GitHub Pages или другом хостинге

## Аутентификация администратора

- **Логин**: sartorial
- **Пароль**: favicon

## Управление объявлениями

### Добавление объявлений
1. Нажмите кнопку "Вход для админов" в заголовке
2. Введите логин и пароль
3. Заполните данные товара (название, описание, цена и т.д.)
4. Выберите категорию товара из списка
5. Введите Telegram-имя продавца (с символом @)
6. Нажмите "Добавить объявление"

### Удаление объявлений
1. Войдите в систему как администратор
2. Перейдите в раздел "Текущие объявления" в панели управления
3. Нажмите кнопку "Удалить" рядом с объявлением, которое нужно удалить

## Варианты развертывания

### GitHub Pages (Рекомендуется)
1. Сделайте форк этого репозитория или создайте свой
2. Включите GitHub Pages в настройках репозитория (Settings → Pages → Source: Deploy from a branch → Branch: main)
3. Ваш сайт будет доступен по адресу `https://yourusername.github.io/repository-name`

### Бесплатные альтернативы
- **Netlify**: Перетащите папку на netlify.com
- **Vercel**: Подключите ваш GitHub репозиторий или используйте `vercel` CLI
- **Firebase Hosting**: Используйте `firebase deploy` после настройки
- **Surge.sh**: Установите surge глобально и выполните `surge` в директории проекта

## Настройка

### Стили
- Цвета можно изменить в CSS переменных `:root`
- Типографику можно изменить, отредактировав импорт шрифтов в `index.html`
- Анимации можно настроить в `script.js` и `styles.css`
