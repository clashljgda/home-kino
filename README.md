# SARTORIAL - Curated Pre-Owned Fashion Marketplace

A minimalist website for a Telegram mini app that sells second-hand luxury fashion items. Features a clean aesthetic inspired by high-end fashion brands like Cévennes and Balenciaga.

## Features

- **Minimalist Design**: Clean, elegant interface with attention to typography and spacing
- **Admin Panel**: Secure admin interface to create new listings
- **Telegram Integration**: Direct connection to sellers via Telegram
- **Responsive Layout**: Works beautifully on all device sizes
- **Animations**: Subtle animations and transitions for enhanced UX
- **Easy Deployment**: Ready for GitHub Pages or similar static hosting

## Deployment Options

### GitHub Pages (Recommended)
1. Fork this repository or create your own
2. Enable GitHub Pages in your repo settings (Settings → Pages → Source: Deploy from a branch → Branch: main)
3. Your site will be live at `https://yourusername.github.io/repository-name`

### Free Hosting Alternatives
- **Netlify**: Drag and drop the folder to netlify.com
- **Vercel**: Connect your GitHub repo or use `vercel` CLI
- **Firebase Hosting**: Use `firebase deploy` after setup
- **Surge.sh**: Install surge globally and run `surge` in the project directory

## Customization

### Adding Listings
1. Click the "Admin" button in the header
2. Fill in the item details (name, description, price, etc.)
3. Enter the seller's Telegram username (with @ symbol)
4. Submit to add the listing

### Styling
- Colors can be adjusted in the CSS `:root` variables
- Typography can be changed by modifying the font imports in `index.html`
- Animations can be customized in `script.js` and `styles.css`

## Technologies Used

- HTML5
- CSS3 (with custom properties)
- Vanilla JavaScript
- Google Fonts (Inter typeface)
